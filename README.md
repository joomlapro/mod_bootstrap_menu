Bootstrap Menu Module for Joomla! 3.0
=====================================

Simple module to show a menu using Twitter Bootstrap for Joomla! 3.0

## Installation

Just download and install.

* [Twitter Bootstrap](http://twitter.github.com/bootstrap/)
